protein-db

An online database of protein isoelectric points

Download .zip Download .tar.gz View on GitHub

Objectives

  • To build a free (as in freedom) web application for viewing protein isoelectric points.
  • To produce a bioinformatics tool with real world value for researchers.
  • The application should provide intuitive but powerful searching facilities.
  • A user account and permissions system will be used to ensure that
  • The application should provide a convenient means for a certified user to edit and upload additional data.
  • The application should present information in a usable and efficient form.
  • Users should be allowed to download generated results for offline use.
  • Adequate security precautions should be made to minimise the risk of important data being sabotaged or stolen.
  • The implementation should use a clean model view controller architecture.
  • Comprehensive test coverage of the API and common use cases should be automated.
  • The application should be fully functional with the provided dataset, but be scalable for much larger datasets.
